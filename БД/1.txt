База данных  -  совокупность взаимосвязанных данных.

Что такое субд?

примеры субд

Это программная реализация технологии хранения, извлечения , обновления и обработки данных в базе данных

иеархическая модель базы данных
это модель базы даннх, где используется представление базы данных в виде древовидной стректуры, состоящей из обзюектов различных уровней
 т.е главный узел => производные

сетевая модел бд -  это модель данных где несколько обхектов могут быть связаны с несколькими обзектами  (отдельные сущности равноправно)

Рельционная бд
все объекты представлены множествами
множества ссылаются на другие множества

master - В этой базе данных хранятся все данные системного уровня для экземпляра SQL Server.

msdb - Используется агентом SQL Server для планирования предупреждений и задач.

model - Используется в качестве шаблона для всех баз данных, создаваемых в экземпляре SQL Server. 

tempdb - Рабочее пространство для временных объектов или взаимодействия результирующих наборов.
